# 惜平凡的组件库

本组件库纯粹就是个人喜好开发，想将自己学到的东西整合起来。

更新随缘

## 安装

### 组件库安装

使用 npm

```bash
$ npm install canvas-painting-dgy
$ npm i
$ npm run dev
```

使用 yarn

```bash
$ yarn add canvas-painting-dgy
$ yarn
$ yarn dev
```

推荐使用 [yarn](https://www.yarnpkg.cn/)，不为什么就是感觉好用。

### 文档开发

在上面的安装结束后

使用 npm

```bash
$ npm run docs:dev
```

使用 yarn

```bash
$ yarn docs:dev
```

## 使用

因为是组件库，所以使用方面在安装好之后先导入相应的组件，然后使用对应标签即可

> 本组件是使用 vue3 编写的，请在 vue3 环境下使用
